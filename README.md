## Judul Website : EcoVibe - Gerakan Hijau untuk Masa Depan Berkelanjutan

## Deskripsi Website : EcoVibe adalah website kampanye lingkungan yang berfokus pada edukasi, dokumentasi, dan ajakan untuk menjaga keberlanjutan alam.
Website ini dirancang dengan Vue.js, TailwindCSS, dan animasi interaktif menggunakan GSAP, sehingga memberikan pengalaman yang informatif sekaligus menarik.

## Fitur Utama:

Home: pengenalan EcoVibe

About: kisah perjalanan ecovibe, nilai - nilai yang menjadi dasar dari gerakan hijau ini. 

Galeri: foto dan dokumentasi kegiatan menjaga kelestarian alam

Contact: formulir kontak untuk saran, kolaborasi, dan diskusi

Responsif: dapat diakses di smartphone, tablet, maupun desktop

Tujuan utama EcoVibe adalah menumbuhkan kesadaran masyarakat akan pentingnya konservasi lingkungan demi masa depan yang lebih hijau dan berkelanjutan.

## Cara Instalasi & Menjalankan Project:
1. Clone Repository,
ketik di terminal : git clone https://github.com/zikran11/webdesign_kawaides_technoversary25.git

2. Masuk ke Folder Project,
ketik di terminal : cd webdesign_kawaides_technoversary25

3. Install Dependencies
Pastikan sudah terpasang Node.js dan npm.
ketik di terminal : npm install

4. Jalankan Project dalam Mode Development,
ketik di terminal : npm run dev
Project akan berjalan pada:
http://localhost:5173

5. Build untuk Production (Opsional)
Jika ingin membuat versi siap deploy,
ketik di terminal : npm run build
